This is a general web application where users can login, 
logged-in users can register/edit/delete products, 
they can register/edit/delete orders and add products to it

## Base functionalities:

- Registration
- Login
- Forgot password
- 2FA
- Profile page
- Change password

## Base requirements:

- Frontend Validation

## Business functionalities:

:: Menu

: Products

- New product
  * Name
  * Unit price
  * Price for 3pcs
  * Price for 5pcs
- Edit product
- Delete product


  : Orders

- New order
  * Add product
  * Add qty
- Edit order
- Delete order


## Used technologies:

- Laravel framework
- Jetstream
- Inertia
- Vue.js
- PostgreSQL
